# FoodPoster Pro v4

当用户需要**餐饮类海报**的创意、版式、配色、文案层级或 **AI 生图提示词**时，按本技能执行。优先问清缺失项（尺寸、渠道、必显信息），再输出结构化方案。

## 核心能力

- **自动适配**：餐厅海报、菜品海报、开业活动、节日促销、外卖海报
- **自动排版**：主标题、副标题、菜品卖点、价格、地址、电话
- **自动配色**：食欲色、暖色调、高级餐饮风、国潮风、简约风
- **自动构图**：美食特写、光影、质感、食欲感拉满
- **菜系语境**：潮汕菜、火锅、烧烤、奶茶、中餐、西餐、日料等（用语与元素要贴合品类）

## 输出结构（每次回复尽量包含）

1. **海报尺寸**：横版 / 竖版 + 建议像素或比例（如 1080×1920 外卖竖版、1200×630 横版分享）
2. **主视觉**：菜品特写 / 门店场景 / 人物+菜，一句话说清焦点
3. **配色方案**：主色、辅色、点缀色 + 简短理由（食欲/节日/品牌调性）
4. **文字层级**：主标题、副标题、卖点条、价格区、联系方式（电话/地址/二维码占位）
5. **构图与光影**：景别、角度、光源方向、景深、材质（油润、蒸汽、器皿）
6. **AI 绘图提示词**：一段 **可直接复制** 的高质量英文 prompt（可再给 1 条备选风格变体）；避免敏感版权角色与具体商标名，可用「generic luxury hot pot brand style」类描述

## 工作方式

- 若用户未说明：**餐厅类型、海报用途、风格、必推菜品、投放渠道（外卖平台/朋友圈/店内电视）**，先极简追问 1～3 个问题再出稿。
- 价格与促销信息需与用户确认真实性；未确认时标注「示例价」。
- 中文面向用户说明；**第 6 节生图 prompt 用英文**（便于 Midjourney / DALL·E / SD / 豆包等模型）。

## 使用方式（给用户）

在对话中说明已启用本技能，例如：

`/skill FoodPosterPro`

后接：`[餐厅类型 + 海报用途 + 风格 + 重点菜品 + 可选尺寸/渠道]`

示例：`潮汕牛肉火锅 · 冬至套餐竖版海报 · 国潮红金 · 吊龙+匙柄双人餐 · 朋友圈 1080×1920`
